BTH RDP FOR iOS

Step 1: Download the Microsoft Remote Desktop app from the App Store

Click to view on App Store

Step 2: Once the app is installed to your iOS device, open the Microsoft Remote Desktop app (usually called RD Client on your iPhone/iPad)

Step 3: The first time you load the application you should follow steps 3-7. You should only need to run through these steps once to get setup. First, under Configuration press on the Settings button

Step 4: From the Settings panel, choose Gateway

Step 5: Now press the Add Gateway button and set the following. Then click the Save button to return to the main screen

Gateway name: BTH Gateway

Server: remoteaccess.bthcc.org.uk

Credentials: Use credentials from connection

Step 6: You will be returned to the Gateway panel where you will see the gateway you have just set up, press on the Settings button in the top left and press Done button (top-right)

Step 7: Press New Remote Desktop and enter the following

Connection name: BTH External Desktop

PC name: bthfarm.bthc.local

Username: bthc\surnamef

Password: ********

Press Gateway button and select: BTH Gateway

Step 8: Under the Remote Desktops, click on BTH External Desktop

Step 9: Press Logon

Step 10: "The publisher can't be identified" - Select "Connect Anyway"

All done!

Important Note: There will be a message saying you are on a temp. account and that your files won't be saved. This is true if you save things in Desktop/Downloads/Documents/etc. but not if you save to your H: or Public drives.
